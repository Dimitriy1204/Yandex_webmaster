     1	# Yandex Wordstat API Agent 📊
     2	
     3	Автоматизированный инструмент для сбора и анализа поискового спроса через **Yandex Search API v2 (Wordstat Dynamics)**. Скрипт собирает помесячную частотность поисковых запросов и формирует детальный Excel-отчёт с аналитикой и визуализацией.
     4	
     5	## 🚀 Возможности
     6	
     7	- ✅ Сбор статистики поисковых запросов через официальный Yandex Search API v2
     8	- ✅ Анализ динамики спроса по месяцам
     9	- ✅ Автоматическая генерация Excel-отчёта с 6 листами:
    10	  - **О файле** — описание отчёта
    11	  - **Данные** — сырые данные по запросам
    12	  - **Сводка** — агрегированная статистика
    13	  - **График** — визуализация трендов
    14	  - **Голубые океаны** — анализ по метрикам momentum, mean, slope (3 блока)
    15	  - **Растущие рынки** — анализ по метрикам momentum, mean, slope (3 блока)
    16	- ✅ Гибкая настройка временного диапазона анализа
    17	- ✅ Поддержка множественных поисковых запросов
    18	- ✅ Логирование и обработка ошибок
    19	
    20	## 📋 Требования
    21	
    22	- Python 3.8+
    23	- API-ключ Яндекс.Облака с доступом к Search API
    24	- Установленные зависимости (см. ниже)

    37	```bash

    51	### 3. Настройка переменных окружения
    52	
    53	Скопируйте файл `.env.example` в `.env`:
    54	
    55	```bash
    56	cp .env.example .env
    57	```
    58	
    59	Заполните `.env` своими данными:
    60	
    61	```env
    62	# API-ключ Яндекс.Облака для Search API (Wordstat)
    63	YANDEX_API_KEY=ваш_api_key_яндекс_облака
    64	
    65	# Каталог в Яндекс.Облаке (folder ID)
    66	FOLDER_ID=ваш_folder_id
    67	
    68	# Поисковые запросы для анализа (через запятую)
    69	SEARCH_QUERIES=купить оборудование для,купить станок для,купить линию для
    70	
    71	# Временной диапазон (опционально)
    72	DATE_FROM=2022-01
    73	DATE_TO=2026-05
    74	```

    76	## 🔑 Получение API-ключа
    77	
    78	1. Перейдите в [Консоль Яндекс.Облака](https://console.cloud.yandex.ru/)
    79	2. Создайте сервисный аккаунт или используйте существующий
    80	3. Назначьте роль `search-api.admin` или `search-api.executor`
    81	4. Создайте статический API-ключ (IAM)
    82	5. Скопируйте ключ в файл `.env`
    83	
    84	**Документация:**
    85	- [Создание API-ключа](https://yandex.cloud/ru/docs/iam/operations/api-key/create)
    86	- [Yandex Search API v2](https://aistudio.yandex.ru/docs/ru/search-api/api-ref/Wordstat/getDynamics)

   143	## ⚙️ Конфигурация
   144	
   145	### Поисковые запросы
   146	
   147	В `.env` укажите интересующие вас запросы через запятую:
   148	
   149	```env
   150	SEARCH_QUERIES=купить диван,заказать пиццу,доставка цветов
   151	```
   152	
   153	### Временной диапазон
   154	
   155	- `DATE_FROM` — начальная дата в формате `YYYY-MM` (по умолчанию: 2022-01)
   156	- `DATE_TO` — конечная дата в формате `YYYY-MM` (по умолчанию: последний полный месяц)
   157	
   158	**Важно:** 
   159	- Статистика Wordstat доступна с задержкой ~1-2 месяца
   160	- Текущий месяц может быть недоступен для анализа
   161	- Указывайте только полные месяцы
   162

   181	## 🛠️ Troubleshooting
   182	
   183	### Ошибка авторизации
   184	
   185	Проверьте правильность `YANDEX_API_KEY` и `FOLDER_ID` в `.env`.
   186	
   187	### Пустые данные в отчёте
   188	
   189	- Убедитесь, что запросы корректны
   190	- Проверьте временной диапазон (не слишком ли давно)
   191	- Убедитесь, что у аккаунта есть права на Search API
   192	
   193	### Ошибка matplotlib
   194	
   195	Установите шрифты:
   196	```bash
   197	sudo apt-get install fonts-dejavu
   198	```
